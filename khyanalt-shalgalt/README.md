# Хяналт шалгалт

Хяналт шалгалт нь дараах үндсэн 4 үе шаттайгаар хэрэгжинэ. Үүнд:

1. **Төлөвлөгөөний  үе шат**
2. **Шалгалтын бэлтгэл үе шат**
3. **Шалгалтын гүйцэтгэлийн үе шат**
4. **Тайлангийн үе шат**

Эдгээр үе шат бүр нь тодорхой зорилго, үйл ажиллагааны дараалал бүхий дэд алхмуудаас бүрдэнэ.   &#x20;
