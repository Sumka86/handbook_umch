# Түрээс

Түрээсийн систем нь тухайн хуулийн этгээд түрээсэлж байгаа болон түрээслүүлэх гэж буй хөрөнгийн сонгон шалгаруулалтын зар, оролцогч нарын мэдээллийг бүртгэнэ. Түрээсийн гэрээний бүртгэлийг хийнэ.Түрээсийн сонгон шалгаруулалтын зарын мэдээллийн болон сонгон шалгаруулалтад оролцогчдын мэдээллийн иргэд олон нийтэд портал системээр дамжуулан нийтэлнэ.
