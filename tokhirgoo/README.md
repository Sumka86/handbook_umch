# Тохиргоо

Тохиргоо цэс нь дараах дэд хэсгүүдээс бүрдэнэ:

* **Системийн хэрэглэгч**\
  Системд нэвтрэх хэрэглэгчийг бүртгэх, эрхийн түвшин тохируулах, мэдээллийг засварлах үйлдлүүдийг гүйцэтгэнэ.
* **Үндсэн байгууллага**\
  Байгууллагын ерөнхий мэдээллийг бүртгэх болон засварлахад ашиглагдана.
* **Дэд байгууллага**\
  Байгууллагын салбар, нэгжүүдийн мэдээллийг бүртгэх, удирдах боломжтой.
* **ӨХБЗ (Өмч хамгаалах байнгын зөвлөл)**\
  Өмч хамгаалах байнгын зөвлөлийн бүрэлдэхүүн болон мэдээллийг бүртгэнэ.
* **Тооллогын комисс**\
  Тооллого хийх комиссын бүртгэл үүсгэх, гишүүдийг удирдах зориулалттай.
* **ТУЗ (Төлөөлөн удирдах зөвлөл)**\
  Төлөөлөн удирдах зөвлөлийн гишүүд болон холбогдох мэдээллийг бүртгэх, засварлахад ашиглана.
* **ХЭЗ (Хувьцаа эзэмшигч)**\
  Хувьцаа эзэмшигчдийн бүртгэл үүсгэх, мэдээллийг удирдах боломжийг олгоно.
