# Удирдлага зохицуулалт

ТӨРИЙН БОЛОН ОРОН НУТГИЙН ӨМЧИТ КОМПАНИЙН УДИРДЛАГА, ЗОХИЦУУЛАЛТЫН СИСТЕМ НЬ ТӨРИЙН БОЛОН ОРОН НУТГИЙН ӨМЧИТ ХУУЛИЙН ЭТГЭЭДИЙН ХУВЬЦАА ЭЗЭМШИГЧДИЙН МЭДЭЭЛЭЛ, ГҮЙЦЭТГЭХ ЗАХИРАЛ БОЛОН ЭРХ БҮХИЙ АЛБАН ТУШААЛТНУУДЫН МЭДЭЭЛЭЛ, ХҮНИЙ НӨӨЦИЙН МЭДЭЭЛЭЛ, ТӨЛӨӨЛӨН УДИРДАХ ЗӨВЛӨЛИЙН ГИШҮҮДИЙН МЭДЭЭЛЭЛ, ТӨРИЙН ӨМЧИТ КОМПАНИУДЫН САНХҮҮ, ЭДИЙН ЗАСГИЙН ҮЗҮҮЛЭЛТҮҮД, ТӨЛӨВЛӨЛТ, ГҮЙЦЭТГЭЛ, ТӨЛӨӨЛӨН УДИРДАХ ЗӨВЛӨЛ БОЛОН ГҮЙЦЭТГЭХ ЗАХИРЛЫН АЖЛЫН ТАЙЛАН, ҮНЭЛГЭЭ, СОНГОН ШАЛГАРУУЛАЛТ, ТЭДГЭЭРИЙН ҮР ДҮН ЗЭРГИЙГ НЭГДСЭН ЦАХИМ УДИРДЛАГАД ОРУУЛЖ, ХЯНАХ, ШИЙДВЭР ГАРГАГЧ БҮХИЙ ОРОЛЦОГЧ ТАЛУУДЫГ МЭДЭЭЛЛЭЭР ХАНГАХ НӨХЦӨЛИЙГ БҮРДҮҮЛЖ БАЙНА
