# СЭЗ төлөвлөлт гүйцэтгэл

Хууль тогтоомжид заасны дагуу  зорилтын санал эсвэл Компанийн  дунд, урт хугацааны бизнес төлөвлөгөөнд туссан 3-5 жилийн санхүү эдийн засгийн үндсэн үзүүлэлт, тухайн жилийн  санхүү, эдийн засгийн үндсэн үзүүлэлт, зорилтот түвшний төлөвлөгөө хэрэгжилтийг бүртгэлжүүлнэ. Санхүү эдийн засгийн төлөвлөлт гүйцэтгэл цэс нь дараах дэд цэсүүдээс бүрдэнэ.
