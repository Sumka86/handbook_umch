# Хүний нөөц

Хүний нөөцийн бүртгэлд хуулийн этгээдийн ажилтнуудын мэдээллийг дараах төрлөөр бүртгэнэ. Үүнд:

* Төлөөлөн удирдах зөвлөлийн гишүүд
* Гүйцэтгэх захирал
* Эрх бүхий албан тушаалтан
* Байгууллагын ажилтан

Тус бүрийг ангиллын дагуу бүртгэж, холбогдох мэдээллийг системд хадгална.
