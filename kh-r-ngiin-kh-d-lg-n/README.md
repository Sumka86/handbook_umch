# Хөрөнгийн хөдөлгөөн

Хөрөнгийн хөдөлгөөн цэс нь хүсэлт, хөрөнгийн жагсаалт, санал болгосон гэсэн дэд цэсүүдтэй.

* Хүсэлт -  ӨХБЗ-ийн хурлын шийдвэрээр байгууллагын дарга руу шилжүүлсэн хүсэлтүүд&#x20;
* Хөрөнгийн жагсаалт - Хүсэлт гаргасан хөрөнгийн жагсаалт
* Санал болгосон - Шийдвэрлэсэн хүсэлтийн хөрөнгийн жагсаалт&#x20;
